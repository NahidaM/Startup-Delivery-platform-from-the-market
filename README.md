# Startup-Delivery-platform-from-the-market
Layihədə çatmaq istənilən məqsəd:
Müştərilərin yaxınlıqdakı supermarketlərə və apteklərə daxil ola biləcəyi və əşyaları evə çatdırmaqla ala biləcəyi vahid platforma (veb səhifə və tətbiqetmə) 
Müştərilərin supermarketlərdən və apteklərdən vaxt itirmədən online əşyalar alaraq onları hazır vəziyyətdə götürə biləcəyi bir platforma (veb səhifə və tətbiqetmə).
